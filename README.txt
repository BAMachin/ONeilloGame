ONeillo Game!

Overview:
The ONeillo Game, is a project created by Bethney Machin from the programming and fundamentals moduale at Sheffield Hallam University (Level 4).
This game is a cumulation of programming skills developed during the course and aims to prove an entertaining experience for the user. 

Game Description:
ONeillo game is a board game based application, where two players will take turns to place counters on the 8x8 game board. 
The aim of the game is to flip the opponents counters and finish the game with the highest number of counters. 

To start playing the game, ensure to click 'Start Game' whether you enter player names or use the default names.
